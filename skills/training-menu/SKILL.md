# 钉钉培训助手 · "你能做什么"菜单

## 触发条件

当用户发送以下任一关键词时触发：
- 你能做什么
- 你能做什么？
- 帮助
- 帮助？
- 菜单
- 菜单？
- 功能
- 有什么功能
- 用法
- 怎么用
- 支持什么
- 可以做什么

**规则：直接输出菜单内容，不要解释、不要额外文字。**

---

## ⚠️ 钉钉 AI Card 限制（重要）

**问题：** 钉钉 AI Card 模板会自动将内容重新分类显示（如"经营分析"、"培训管理"、"行政支持"），**覆盖**技能定义的菜单格式。

**解决：** 触发此技能时，消息必须走**普通 Markdown** 路径，**禁用 AI Card**。

### 发送方式

| 场景 | 发送方式 | 说明 |
|------|----------|------|
| **Kimi 私聊** | 正常回复 | 显示标准菜单格式 ✅ |
| **钉钉群@** | 需设置 `useAICard: false` | 否则显示 AI Card 预设分类 ❌ |
| **钉钉私聊** | 需设置 `useAICard: false` | 否则显示 AI Card 预设分类 ❌ |

### 技术说明

在钉钉 connector 中发送消息时，需要明确设置参数：

```javascript
// 错误：使用 AI Card（内容会被模板覆盖）
await sendProactive(config, target, content, {
  useAICard: true  // ❌ 不要这样
});

// 正确：使用普通 Markdown 消息
await sendProactive(config, target, content, {
  useAICard: false,  // ✅ 禁用 AI Card
  msgType: 'markdown'
});
```

### 当前限制

由于 OpenClaw 核心层在钉钉群中默认启用 AI Card，**此技能在钉钉群中的回复格式可能无法完全控制**。

**临时解决方案：**
1. 在 Kimi 中询问"你能做什么" → 显示标准格式
2. 在钉钉群中询问 → 可能显示 AI Card 格式（经营分析/培训管理/行政支持）
3. 如需统一格式，需要修改 OpenClaw 钉钉 connector 的默认配置

## 标准回复格式（钉钉专用）

### 格式要求（严格遵守）

| 要求 | 说明 | 错误示例 | 正确示例 |
|------|------|----------|----------|
| **空行分段** | 钉钉会把连续行合并，必须用空行隔开 | `1. 订货？ 2. 班前会？` | `1. 订货？`（空行）`2. 班前会？` |
| **数字序号** | 必须用 `1. 2. 3.`，不要用 `•` 或 `-` | `• 怎样更准确订货？` | `1. 怎样更准确订货？` |
| **标题空行** | 标题和第一个项目之间必须空一行 | `**一、辅导** 1. 订货？` | `**一、辅导**`（空行）`1. 订货？` |
| **称呼+名字** | 称呼用钉钉上的姓名，不要姓 | `陈晓腾，你好` | `晓腾，你好呀` |
| **每行一个** | 每个项目单独一行，不要挤在一起 | `1. 订货？ 2. 班前会？` | `1. 订货？`（空行）`2. 班前会？` |
| **结尾引导** | 加一句引导语+emoji，最后加@提示 | | `需要我干什么...` |

### 标准回复内容

```
{姓名}，你好呀 👋 我不是客服，不是搜索引擎，是懂聚满分营运和培训的AI助教 🤖。我不仅可以解答问题，还可以动手帮你干活。有关培训、门店经营事情都可以找我聊聊 💬。

**【解疑答惑】**

**一、学习内容辅导**

1. 怎样更准确订货？

2. 班前会考哪些内容？

3. 怎样改善门店毛利率？

4. 如何赢得周周pk？

5. 刚做领班，员工不听我安排怎么办？

6. 点评我的学习总结

7. 任何其他学习问题

**二、师傅带训咨询**

1. 怎样才能帮学员100%通过考核？

2. 没时间带训怎么办？

3. 需要完成哪些工作才能申请考核？

4. 任何其他师傅带训问题

**【动手干活】**

1. 帮我整理报销单

2. 帮我做损益计划

3. 帮我做营业额计划

4. 帮我做高峰节假计划

5. 帮我做挑战日计划

需要我干什么，直接跟我说就行，比如："帮我做挑战日计划" 📝
**在群里面@我，或私聊都可以**
```

### 注意事项

1. **姓名处理**：从钉钉消息中提取 sender 的昵称，去掉姓氏。如果无法提取昵称，用"你好呀"开头。
2. **不要添加**："以上是菜单"、"请查看"、"这是我能做的"等额外说明文字。
3. **不要添加**： emoji 过多（正文已有 👋 🤖 💬 📝，不要再加）。
4. **【动手干活】**中的技能列表随技能扩展而更新，当前为 5 项。
5. **结尾必须加**：`在群里面@我，或私聊都可以`，加粗格式。
